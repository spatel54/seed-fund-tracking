# Visualizations

## Overview
Static PNG charts and interactive HTML dashboards for IWRC Seed Fund data visualization.

## Contents

### [Static Visualizations](static/) (36 PNG files)
High-resolution charts organized by category:
- **overview/** - Investment, ROI, and project trends
- **institutions/** - Institutional reach and funding distribution
- **students/** - Student training outcomes
- **topics/** - Research topic area analysis
- **awards/** - Award type comparisons (all-time, 10-year, 5-year)
- **project_types/** - Project type breakdowns (104B-only and all projects)

### [Interactive Dashboards](interactive/) (15 HTML files)
Web-based interactive visualizations:
- **core/** - Main dashboards (ROI, detailed analysis, investment, students, projects)
- **geographic/** - Illinois institutions maps
- **award_types/** - Award type analysis dashboards
- **project_types/** - Interactive project type breakdowns

## Technical Specs

**Static (PNG):**
- Resolution: 300 DPI
- Format: PNG with transparency
- Suitable for: Presentations, reports, print

**Interactive (HTML):**
- Technology: Plotly.js, Folium (maps)
- Browser: Chrome 90+, Firefox 88+, Safari 14+
- Internet: Required for map tiles
