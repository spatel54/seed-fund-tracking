## üöÄ How to Use This Export

### For Data Review
1. Unzip the package
2. Open `data/consolidated/IWRC Seed Fund Tracking.xlsx` in Excel
3. Review 539 projects with 35 metrics per project

### For Analysis Review
1. Open `REPOSITORY_SUMMARY.md` for complete overview
2. Open `docs/RESULTS_READY.md` for latest findings
3. Open `docs/ANALYSIS_SUMMARY.md` for detailed insights

### For Visualizations
**Static Images:**
- Open any PNG file in `visualizations/static/`
- All images are 300 DPI, ready for print/presentations

**Interactive Charts:**
- Open any HTML file in `visualizations/interactive/`
- Works in any modern web browser
- Hover for details, click to zoom, drag to pan

### For Notebooks
1. Install Jupyter: `pip install jupyter pandas openpyxl matplotlib seaborn plotly`
2. Open any `.ipynb` file from `notebooks/current/`
3. Run cells to reproduce analysis
