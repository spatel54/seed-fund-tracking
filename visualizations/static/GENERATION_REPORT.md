## Methodology Notes

### Project Counting
- Years extracted from Project ID using regex pattern `(20\d{2}|19\d{2})`
- Unique projects counted using `df['Project_ID'].nunique()`
- Corrects previous methodology that used row counts (`len(df)`)

### Follow-on Funding Extraction
- Parses monetary values from multiple text columns
- Extracts dollar amounts using regex pattern `\$?\s*([\d,]+(?:\.\d{2})?)`
- Sums values from:
  - Monetary Benefit column
  - Award Description column
  - Award/Grant text

### ROI Calculation
- ROI = Follow-on Funding / IWRC Investment
- 10-Year: $275,195 / $8,516,278 = 0.03x
- 5-Year: $261,000 / $7,319,144 = 0.04x
