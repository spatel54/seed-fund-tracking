# Interactive Dashboards - Viewing Guide

## Quick Start

All interactive HTML files can be opened directly in any web browser without requiring installation or internet connection.

### Option 1: Direct Open (Easiest)
1. Navigate to the `visualizations/interactive/` folder
2. Double-click any `.html` file
3. File opens automatically in your default browser

### Option 2: Drag & Drop
1. Open your web browser (Chrome, Firefox, Safari, Edge)
2. Drag an `.html` file into the browser window
3. Dashboard loads instantly

### Option 3: Right-Click Menu
1. Right-click an `.html` file
2. Select "Open With" → Choose your browser
3. Dashboard opens

## How to Use Interactive Features

### Hover Tooltips
Move your mouse over data points, bars, or regions to see:
- Institution names
- Funding amounts
- Project counts
- Additional metrics
- Percentages and calculations

**Example:** Hover over a bar in a chart to see the exact value

### Zoom & Pan
Many dashboards support zooming:
- **Scroll wheel:** Zoom in/out on maps
- **Click & drag:** Pan around the map
- **Double-click:** Reset to default view
- **Box select:** Some charts allow selecting data ranges

### Click for Details
- **Click institutions:** Get detailed information
- **Click chart elements:** Filter or highlight related data
- **Click legend items:** Show/hide data series

### Search & Filter
Some dashboards include search fields:
1. Type institution name or search term
2. Results update in real-time
3. Charts adjust to show filtered data

## Troubleshooting

### Dashboard Won't Open
- **Solution:** Try a different browser
- **Solution:** Check if JavaScript is enabled
- **Solution:** Move file closer to root directory (avoid long paths)

### Charts Not Displaying
- **Solution:** Refresh the page (F5 or Cmd+R)
- **Solution:** Clear browser cache
- **Solution:** Try a different browser
- **Solution:** Ensure file wasn't corrupted during transfer

### Slow Performance
- **Solution:** Close other browser tabs
- **Solution:** Restart browser
- **Solution:** Use Chrome or Firefox
- **Solution:** Check available RAM

### Interactive Features Not Working
- **Solution:** Enable JavaScript in browser settings
- **Solution:** Clear browser cache and cookies
- **Solution:** Try a different browser
- **Solution:** Update to latest browser version

## File Management

### Copying/Backing Up
Files can be:
- Copied to USB drives
- Emailed to colleagues
- Shared via cloud storage (Google Drive, OneDrive, Dropbox)
- Printed to PDF from browser (Print → Save as PDF)

### File Organization
Keep files in the same folder structure for best results:
```
visualizations/
├── interactive/
│   ├── index.html
│   ├── institutional_distribution_map.html
│   ├── roi_analysis_dashboard.html
│   └── [other HTML files]
└── static/
    └── [PNG charts]
```

### Renaming Files
- Files can be renamed without affecting functionality
- Keep `.html` extension
- Avoid special characters in filename
