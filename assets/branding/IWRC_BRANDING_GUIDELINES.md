- **Uppercase:** ABCDEFGHIJKLMNOPQRSTUVWXYZ
- **Lowercase:** abcdefghijklmnopqrstuvwxyz
- **Numbers:** 0123456789

## Usage Notes

- Use Montserrat Semibold for all headlines and titles
- Use Montserrat Light for body text and descriptions
- Primary color (#258372) should be used for main branding elements
- Secondary color (#639757) for supporting elements
- Accent color (#FCC080) for highlights and calls-to-action
- Maintain tagline visibility across all materials
