# IWRC Logos

## Available Formats

### PNG Format
**File:** `IWRC_Logo.png`
- **Size:** 4 KB
- **Format:** Raster (PNG)
- **Use Cases:** Web pages, presentations, documents, social media

### SVG Format
**File:** `IWRC_Logo.svg`
- **Size:** 2.6 KB
- **Format:** Vector (SVG)
- **Use Cases:** Print materials, scalable web graphics, high-resolution displays

## Usage Guidelines

### When to Use PNG
- Web pages and HTML dashboards
- PowerPoint presentations
- Microsoft Word documents
- Email signatures
- Social media profile images
- Quick mockups and prototypes

### When to Use SVG
- Print materials (brochures, posters, reports)
- High-resolution displays (Retina, 4K monitors)
- Scalable web graphics that need to resize
- Professional graphic design work
- Vector editing software (Adobe Illustrator, Inkscape)

## Best Practices

### Size and Spacing
- Maintain minimum clear space around logo equal to the height of the "IWRC" text
- Never stretch or distort the logo
- Minimum width: 100px for web, 0.5 inches for print

### Color Usage
- Logo uses IWRC brand colors (see IWRC_BRANDING_GUIDELINES.md)
- Always use on white or light backgrounds for best visibility
- Ensure sufficient contrast with background

### Don'ts
- ❌ Don't change logo colors
- ❌ Don't add effects (shadows, outlines, gradients)
- ❌ Don't rotate or skew the logo
- ❌ Don't place on busy backgrounds
- ❌ Don't recreate or modify the logo design

## File Specifications

### IWRC_Logo.png
- Transparent background
- Optimized for web
- RGB color space
- 72-150 DPI suitable for screen display

### IWRC_Logo.svg
- Vector scalability (infinite resolution)
- Embedded fonts/outlines
- RGB color space
- Can be converted to CMYK for print
