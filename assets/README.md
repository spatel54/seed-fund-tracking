## Folder Descriptions

### branding/
Contains all IWRC brand identity resources including logos, fonts, and comprehensive branding guidelines.

#### branding/logos/
IWRC logo files in multiple formats:
- `IWRC_Logo.png` - PNG format (4 KB) for web and documents
- `IWRC_Logo.svg` - Vector SVG format (2.6 KB) for scalable use

See [logos/README.md](branding/logos/README.md) for detailed usage guidelines.

#### branding/fonts/
Official IWRC typeface - Montserrat font family:
- `Montserrat-Regular.ttf` - Regular weight (323 KB)
- `Montserrat-Bold.ttf` - Bold weight (328 KB)

See [fonts/README.md](branding/fonts/README.md) for font information and usage.

### styles/
Shared CSS stylesheets for maintaining consistent IWRC branding across web pages and dashboards.

## Brand Colors

**Primary Colors:**
- **Primary Teal:** #258372
- **Secondary Olive:** #639757

**Supporting Colors:**
- **Text Gray:** #54595F
- **Accent Peach:** #FCC080
- **Background Light Gray:** #F6F6F6

See [branding/IWRC_BRANDING_GUIDELINES.md](branding/IWRC_BRANDING_GUIDELINES.md) for complete brand specifications.

## Usage

All visualizations, reports, and web pages should follow the IWRC branding guidelines. Use the provided logos, fonts, and color palette to maintain brand consistency.

## Dependencies

- Montserrat fonts should be embedded in web pages or included in PDF generation
- SVG logo preferred for high-resolution displays and print materials
- PNG logo suitable for web and presentations
