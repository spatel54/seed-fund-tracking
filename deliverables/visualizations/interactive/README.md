# Interactive Visualizations

## Overview
15 interactive HTML dashboards with hover tooltips, zoom, pan, and filtering capabilities.

## Categories

### core/ (5 dashboards)
Main analysis dashboards:
- **roi_analysis_dashboard.html** - ROI trends and metrics
- **detailed_analysis.html** - Multi-panel comprehensive view
- **investment_interactive.html** - Investment treemap
- **students_interactive.html** - Student distribution sunburst
- **projects_timeline.html** - Chronological project timeline

### geographic/ (2 maps + guide)
Illinois institutions mapping:
- **institutional_distribution_map.html** - Basic institutions map
- **institutions_map_enhanced.html** - Enhanced map with detailed popups
- **MAP_GUIDE.md** - Map usage documentation

### award_types/ (3 dashboards)
Award type analysis:
- **award_type_dashboard.html** - Award type overview
- **award_type_comparison.html** - Comparative analysis
- **award_type_deep_dive.html** - Detailed breakdown

### project_types/ (4 dashboards)
Project type breakdowns:
- **interactive_roi_dashboard.html** - ROI by project type
- **interactive_topic_distribution.html** - Topic distribution
- **project_type_breakdown_104b_only.html** - 104B projects only
- **project_type_breakdown_all_projects.html** - All projects

### Navigation Hub
- **index.html** - Main dashboard navigation (in parent folder)

## Features
- Hover tooltips with detailed information
- Interactive legends (click to show/hide)
- Zoom and pan controls
- Download as PNG
- Responsive design

## Browser Requirements
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Internet connection (for map tiles and CDN)

## Usage
Open HTML files directly in a web browser. No server required.
