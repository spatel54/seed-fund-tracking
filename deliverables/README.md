# Deliverables

## Overview
This folder contains all final outputs from the IWRC Seed Fund Tracking & Analysis System, including PDF reports, static visualizations (PNG charts), and interactive HTML dashboards.

## Quick Access

- **[Reports](reports/)** - PDF reports and summaries
- **[Visualizations](visualizations/)** - Static PNG charts and interactive HTML dashboards
- **[Navigation Hub](index.html)** - Interactive interface to browse all deliverables

## Contents Summary

### Reports (6 PDFs)
- **Executive Reports** (3): Executive summary, fact sheet, financial summary
- **Detailed Analysis** (3): Comprehensive analysis, project type breakdowns

### Visualizations
- **Static Charts** (36 PNGs): Organized by category (overview, institutions, students, topics, awards, project types)
- **Interactive Dashboards** (15 HTML): Core dashboards, geographic maps, award analysis, project type breakdowns

## Data Period

- **Years Analyzed:** 2015-2024 (10-year period)
- **Projects:** 77 unique projects
- **Total Investment:** $8.5 million
- **Students Supported:** 304

## File Naming Conventions

All files follow consistent naming:
- Lowercase with underscores
- Descriptive names indicating content
- Time periods specified (e.g., `_10_year`, `_all_time`)
- Project types indicated (e.g., `_104b_only`, `_all_projects`)

## Usage

### Accessing Reports
- Navigate to [reports/](reports/) folder
- Executive summaries in `executive/` subfolder
- Detailed analysis in `detailed/` subfolder

### Viewing Visualizations
- **Static Charts:** Browse categorized folders in [visualizations/static/](visualizations/static/)
- **Interactive Dashboards:** Open HTML files in [visualizations/interactive/](visualizations/interactive/) with any modern web browser

### Navigation Interface
- Open [index.html](index.html) for a visual navigation hub
- Browse by category or search for specific deliverables

## Technical Notes

- **PDF Reports:** Compatible with all PDF readers (Adobe Acrobat, Preview, etc.)
- **PNG Charts:** 300 DPI, suitable for presentations and print
- **HTML Dashboards:** Require internet connection for map tiles and CDN resources
- **Browser Compatibility:** Chrome 90+, Firefox 88+, Safari 14+, Edge 90+
