## To Push to GitHub

### Step 1: Create GitHub Repository
1. Go to https://github.com/new
2. Create a new repository (e.g., "seed-fund-tracking")
3. Do NOT initialize with README (we already have one)
4. Copy the repository URL

### Step 2: Configure Remote
```bash
cd "/Users/shivpat/Downloads/Seed Fund Tracking"
git remote add origin <PASTE_YOUR_GITHUB_URL>
```

Example:
```bash
git remote add origin https://github.com/your-username/seed-fund-tracking.git
```

### Step 3: Push to GitHub
```bash
git push -u origin main
```

### Step 4: Verify
```bash
git remote -v
```

## Useful Git Commands

### View Status
```bash
git status

```

### Create and Push Feature Branch
```bash
git checkout -b feature/my-feature
git add <files>
git commit -m "Description"
git push -u origin feature/my-feature
```

### Pull Latest Changes
```bash
git pull origin main
```
